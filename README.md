# 田原さんのお家見守りサービス

高松市内限定の空き家見守りサービスのランディングページです。

## サービス概要

- **サービス開始**: 2025年12月予定
- **対象エリア**: 高松市内限定
- **モニター募集**: 限定5軒

## プラン

### ライトプラン（屋外のみ）
- 料金: 4,800円/月
- モニター特典: 半年間50％OFF（2,400円/月）

### スタンダードプラン（屋内＋屋外）
- 料金: 8,800円/月
- モニター特典: 半年間50％OFF（4,400円/月）

## モニター特典

- ✅ 半年間50％OFF
- ✅ 短期契約が可能（6ヶ月から）
- ✅ 緊急巡回1回無料
